# Customer Churn Prediction - Feature Engineering Documentation

## Project Title

**Customer Churn Prediction using Data Preprocessing & Feature Engineering**



# Introduction

Feature engineering is the process of creating new features from existing data to improve the performance of machine learning models. Well-designed features help the model learn important patterns and make more accurate predictions.



# Objective

The main objective of feature engineering is to transform the original dataset into a more useful format by creating meaningful features that improve prediction accuracy.
